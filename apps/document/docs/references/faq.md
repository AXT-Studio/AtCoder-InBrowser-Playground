# FAQ

以下に、よくある質問とその回答をまとめます。

:::details AIBPは何をする拡張機能ですか？
AIBPは、AtCoderの問題ページにコードエディターとコードテスターを追加する拡張機能です。
AIBPを導入すると、AtCoderの問題ページから移動することなくコードを編集し、テストを行い、提出することができます。
詳しい使い方などは[ガイド](./../guides/index.md)をご覧ください。
:::

:::details どのブラウザで使えますか？
- FirefoxとChromeでは確実に使うことができます。
- EdgeやOpera、Brave、Floorpなどでも使うことができるはずです。
- Safariで使うことはできません。
:::

:::details インストールしたのに表示されません。
- AtCoderのコンテストの問題ページ以外でAIBPのUIは表示されません。問題ページを開いてください。
- 問題ページを開いている場合は、一度ページをリロードしてみてください。
- 拡張機能が有効化されているかを、ブラウザの設定から確認してみてください。
- それでもだめならバグかもしれません。「バグを見つけたらどこに報告すればいいですか？」を参考に開発者に報告してくれると嬉しいです。
:::

:::details どのページで使えますか？
AIBPは、AtCoderのコンテストの問題ページで使えます。
より具体的には、URLが`https://atcoder.jp/contests/*/tasks/*`という形のページで使用できます。
:::

:::details AIBPのコードテストは、コンテストの「コードテスト」ページや「提出」と何が違いますか？
AIBPのコードテストは、コンテストの「コードテスト」ページからのテスト実行を、AtCoderのサーバーではなくあなた自身のパソコンで行うようなものです。
|      | AIBPのコードテスト | コンテストの「コードテスト」ページ | 提出 |
|-----:|:-------------------|:-----------------------------------|:-----|
| どこで実行される？ | あなたのパソコン | AtCoderのジャッジサーバー | AtCoderのジャッジサーバー |
| 得点獲得・ペナルティの対象になる？ | ならない | ならない | なる |

なお、あなた自身のパソコンでコードテストを行う都合上、AIBPのコードテストはAtCoderのジャッジサーバーと一部異なる挙動を示します。
詳細については、[対応環境](./supported.md#対応環境)の各言語のセクションをご覧ください。
:::

:::details 対応言語は何ですか？
JavaScript, TypeScript, Python, Textに対応しています。
詳細については、[対応環境](./supported.md#対応環境)をご覧ください。
:::

:::details 実行結果が AtCoder と違うことがありますか？
はい、一部のコードがAtCoderのジャッジサーバーと異なる挙動を示します。
詳細については、[対応環境](./supported.md#対応環境)の各言語のセクションをご覧ください。
:::
