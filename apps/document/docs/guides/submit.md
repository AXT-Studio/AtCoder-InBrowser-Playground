# 3. 提出する

書いたコードが正しいという確信を持てたら、コードを提出しましょう！
Operationタブの「Prepare Submission」ボタンを使うことで、簡単に提出を行うことができます。

![OperationタブのUI](/tab_operation.webp)

「Prepare Submission」ボタンを押すと、以下の処理が行われます。

- 現在コードエディターに書いてあるコードを、問題ページ下部のソースコード入力欄に自動でコピーします。
- ソースコード入力欄がある位置まで自動画面をでスクロールします。

この機能を使うと、コードエディタ欄に書いているコードの提出を以下の3クリックで行うことができます。

1. 「Operation」タブを開く
2. 「Prepare Submission」ボタンを押す
3. 提出ボタンを押す
