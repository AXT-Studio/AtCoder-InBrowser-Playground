TypeScript向けの機能追加を行った、ver1.4.0をリリースしました。

## 変更点

### feat(TypeScript): InputScannerテンプレートの機能追加 ([#76](https://github.com/AXT-Studio/AtCoder-InBrowser-Playground/issues/76))

TypeScript向けテンプレート「TypeScript (InputScanner)」のコードに含まれている`InputScanner`クラスについて、「`g`個周期で振り分けることを`n`回繰り返す」機能を追加しました。

たとえば、$N$頂点$M$辺の重み付きグラフが以下のように与えられるような場合を考えます。

```
N M
u_1 v_1 w_1
u_2 v_2 w_2
︙
u_M v_M w_M
```

これまでは、たとえば以下のように書く必要がありました。

```ts
const [N, M] = scanner.int(2);
const u: number[] = [];
const v: number[] = [];
const w: number[] = [];
for (let i = 0; i < M; i++) {
    u.push(scanner.int()!);
    v.push(scanner.int()!);
    w.push(scanner.int()!);
}
```

ver1.4.0以降に付属する`InputScanner`では、以下のように書くことができます。

```ts
const [N, M] = scanner.int(2);
const [u, v, w] = scanner.number(M, 3);
```
