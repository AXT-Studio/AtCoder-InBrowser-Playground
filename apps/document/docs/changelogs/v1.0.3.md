## 変更点

### fix: Chrome(MV3)環境で、一部ユーザーがテスト実行に失敗する問題([#54](https://github.com/AXT-Studio/AtCoder-InBrowser-Playground/issues/54))を修正

[ver1.0.2](./v1.0.2.md)で行った修正について、その修正方法として使った機能が現在のChromeのバージョンでは「段階的リリース」の対象だったため、まだリリースされていないユーザーの環境でエラーとなる問題を修正しました。
