## 変更点

### esbuild-wasm, core-jsの導入

QuickJSで最新のJavaScript・TypeScriptを実行するために、esbuild-wasmとcore-jsを導入しました。
ES2025までの機能は概ねサポートされているはずです。

### ドキュメントの新設

今まさにあなたが閲覧している、ドキュメントを新設しました。

### Chrome Web Storeでの公開

新たに、Chrome Web Storeでの公開を開始しました。
